## Install

Requirements: `ruby` in PATH, and have installed gems [rbs][] and [typeprof][].

1. Open package control by pressing `ctrl+shift+p`,

2. `Package Control: Add Repository`, paste `https://github.com/hyrious/sublime-rbs` and return.
