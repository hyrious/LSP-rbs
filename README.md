[ruby/rbs](https://github.com/ruby/rbs).

## Install

Requirements: `ruby` and `rbs` in PATH.

1. Open Package Control by pressing `ctrl+shift+p`
2. `Package Control: Add Repository`, paste
   ```
   https://github.com/hyrious/sublime-rbs
   ```
